Portals in Godot

Custom portal implementation supporting recursive rendering, seamless teleportation, and object preservation through portals. Features real-time render texture updates and physics interaction.

Dynamic Gravity System

Outer Wilds-inspired gravity system featuring spherical gravity fields, multiple gravity sources, and seamless transitions between them.

Includes orbit mechanics, gravity scaling, and proper character controller orientation with smooth rotation transitions.

Game Jam Project

The Stuck

A time-loop based puzzle game where you must save a town from an impending volcano eruption. Each loop provides new information and opportunities to change the outcome.

Developed for GMTK 2025 game jam, featuring dynamic time mechanics, branching narratives, and environmental storytelling.
